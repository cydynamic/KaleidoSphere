# Self-Dev 开发指南

这里用于规定如何开发万花境自身。

适用范围：
- 仅适用于本仓库的系统本体与维护文档。
- 实例知识库维护在 `docs/project-shared/`。

自举启动：
- 对助手说：“请读取 `docs/start.md`，帮我开发万花境。”
- 不执行 `.kaleidosphere` 的更新拉取流程，本仓库即系统本体。

一致性规则：
- 修改 `docs/install.md` 或启动器内容时，必须同步更新本文件的自举说明。
- 修改内置流程文档时，必须同步更新流程决策能力/技能。

测试：
- 参考 `meta-studio/self-dev/tests/README.md`。
