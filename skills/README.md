Skills are modular capabilities that define **WHAT** the Agent can do.

## Directory Structure

*   **[Core Skills](core/)**: Built-in, essential skills for the development lifecycle.
*   **[External Skills](external/)**: Third-party integrations and specialized tools.

## Usage

To activate a skill, ask the Agent to read the corresponding `SKILL.md` file.
The Agent must discover skills from both `skills/core/` and `skills/external/`.

> "Please read `skills/core/coding/SKILL.md` and implement the login feature."
